RPIPE: Biblography Services

Service 1: DBLP Importing

1. type a DBLP URL for person or proceeding url: e.g.
Demo1: use DBLP author page: http://www.informatik.uni-trier.de/~ley/db/indices/a-tree/d/Ding:Li.html
Demo2: use DBLP bibtex page: http://dblp.uni-trier.de/rec/bibtex/conf/www/DingTM08
Demo3: use DBLP proceedings page: http://www.informatik.uni-trier.de/~ley/db/conf/semweb/iswc2008.html
2. Set options please set tag(s) for the imported publication(separated by ,), e.g. "semantic web, provenance":

output output_option:  
3. submit
Note: once the output has been generate, you may save it and import the dump using special:Import feature of wiki.

description

We offer a REST services for converting an authors' publications information on DBLP into various formats
  • xml: this allows any xslt translation to pipe on it
  • bibtex: this aggregates bibtex of all publications on one page
  • i.publication: this generates an xml dump for a set of wiki pages, you may than import them into your wiki.
