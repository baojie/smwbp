description

We offer a REST services for converting an authors' publications information on DBLP into various formats
  • xml: this allows any xslt translation to pipe on it
  • bibtex: this aggregates bibtex of all publications on one page
  • i.publication: this generates an xml dump for a set of wiki pages, you may than import them into your wiki.

This service is a free community service to the public Web. The data is obtained from DBLP.
